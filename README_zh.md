# 🌅 夸父 · 认知框架蒸馏引擎

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> 「追的不是太阳，是太阳背后的运行规律。框架能提取，灵感不能；但提取了框架，灵感的概率就会上升。」

</div>

---

## 夸父是什么？

夸父是一个 AI Skill，能将任何人、流派或思想体系蒸馏成**可运行的认知操作系统**。它从 [女娲](https://github.com/alchaincyf/nuwa-skill) 进化而来，关键新增了**元认知层**——让蒸馏出的框架是动态系统，而非静态快照。

**核心区分**：我们捕捉的是 *他们如何思考*，不是 *他们说了什么*。

---

## 六层蒸馏框架

| 层次 | 内容 | 关键问题 |
|------|------|---------|
| Layer 1 | 表达DNA | 他的句式、节奏、禁忌词是什么？ |
| Layer 2 | 心智模型 | 他用什么镜片看世界？ |
| Layer 3 | 决策启发式 | 他的直觉规则是什么？ |
| Layer 4 | 反模式 | 他的价值观底线在哪？ |
| Layer 5 | 诚实边界 | 这个框架在哪里会失效？ |
| Layer 6 ★ | 元认知层 | 他如何更新自己的认知系统？ |

**Layer 6 是夸父相对女娲的核心进化**：没有元认知层，蒸馏出来的是静态快照；有了它，才是动态认知系统。

## 快速开始

### 安装

```bash
# 将 SKILL.md 复制到你的 skills 目录
cp SKILL.md ~/.config/opencode/skills/kuafu-framework-distiller/SKILL.md
```

### 使用

直接输入人名或思想体系即可调用：

```
/distill 查理·芒格
/distill 塔勒布的反脆弱框架
/distill 我需要一个能帮我做决策的思维框架
```

## 执行流程

### Phase 0：入口分流

| 用户输入 | 路径 |
|---------|------|
| 明确的人名/流派/思想体系 | **直接路径** → Phase 0A |
| 模糊的需求/困惑 | **诊断路径** → Phase 0B |

### Phase 1：信息采集

- **本地语料模式**：用户提供的文件（书籍、演讲、博客）优先级最高
- **网络搜索模式**：系统性搜索著作、访谈、社交媒体、传记

### Phase 2：六层蒸馏

每层都经过严格验证：
- 心智模型需通过**三重验证**：跨域复现、生成力、排他性
- 决策启发式需具备：触发条件 → 规则 → 底层逻辑 → 已知反例

### Phase 3：压力测试（红蓝军）

每个核心模型都面对：
- 🔵 **Blue Team**：最优场景和成功案例
- 🔴 **Red Team**：系统性失效模式和历史反例

### Phase 4：生成 SKILL.md

输出遵循标准结构，存储于：
```
.claude/skills/[人名拼音或英文]-perspective/SKILL.md
```

## 输出结构

生成的 SKILL.md 包含：

```
├── 角色扮演规则（第一人称激活）
├── Layer 1：表达DNA
├── Layer 2：心智模型库（3-7个，三重验证）
├── Layer 3：决策启发式手册
├── Layer 4：反模式清单
├── Layer 5：诚实边界
├── Layer 6：元认知层 ★
│   ├── 框架更新机制
│   ├── 内在张力图谱
│   └── 认知演化轨迹
└── 回答工作流（Agentic Protocol）
```

## 特殊场景

### 主题Skill（非人物）

当需求指向某个领域的方法论而非某个人时：

| 维度 | 人物Skill | 主题Skill |
|------|----------|----------|
| 核心 | 一个人的思维方式 | 一个领域的思维工具箱 |
| 模型来源 | 主要来自一人 | 综合多人视角 |
| 表达风格 | 模拟此人表达 | 中性但专业 |

### 更新已有Skill

```
/update 查理·芒格的skill，补充2024年最新访谈内容
```

### 处理信息匮乏

对于公开信息极少的对象：
- 明确标注低置信度
- 仍完成六层蒸馏，但加注说明
- 在诚实边界中列出信息缺口

---

## 诚实边界（夸父自身的局限）

- 蒸馏不了**直觉**——框架能提取，灵感不能
- 捕捉不了**突变**——只是调研时间节点的快照
- 公开表达 ≠ 真实想法——只能基于可观测信息
- **元认知层是推断性的**——此人未必能自我描述框架更新机制
- **行动建议有时效性**——市场/政策/技术环境变化后，建议需重新验证

> *一个不告诉你局限在哪的Skill，不值得信任。*

## 示例

### 蒸馏一个人物

```
输入：蒸馏查理·芒格的投资哲学

输出：包含以下内容的 SKILL.md
- 5个心智模型（逆向思维、能力圈、多元思维模型等）
- 8条决策启发式，带触发条件
- 元认知：芒格如何通过「反转」和阅读传记更新认知
- 张力图谱：耐心 vs 果断（高张力）
```

### 蒸馏一个主题

```
输入：创建一个理解复杂系统的框架

输出：包含以下内容的 SKILL.md
- 来自塔勒布、圣塔菲研究所、演化经济学的心智模型
- 处理非线性问题的决策启发式
- 元认知：该领域的范式如何演化
```
